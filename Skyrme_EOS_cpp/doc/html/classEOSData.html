Detailed Description

Class for storing state data passed to or returned from equations of state. Performs various error checking to make sure that values being asked for are have been set. If the variable asked for is not set, a std::logic_error is thrown. This class provides a uniform, extensible interface to EoSs such that new variables can easily be added. All inputs should assume hbar = c = 1 and be expressed in terms of fm.

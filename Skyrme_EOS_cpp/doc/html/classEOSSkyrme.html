Detailed Description

Implements a uniform matter, Skyrme type EOS for with arbitrary parameters.

std::vector< EOSData > EOSSkyrme::FromMuAndT ( const EOSData &  eosIn) const
virtual

Get the state of the material from the temperature and the neutron and proton chemical potentials. Since there may be multiple points in the density temperature space which have a given set of chemical potentials the a vector of EOSData is returned.
